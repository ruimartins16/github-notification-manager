# GitHub Pages Setup Instructions

This directory hosts the privacy policy for the Chrome Web Store listing.

## Setup GitHub Pages

1. Go to your GitHub repository settings
2. Navigate to **Settings** â†’ **Pages**
3. Under "Source", select:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/docs`
4. Click **Save**

## Access URL

After setup (takes a few minutes), your privacy policy will be available at:

```
https://[your-username].github.io/github-notification-manager/privacy-policy.html
```

Or for markdown:
```
https://[your-username].github.io/github-notification-manager/privacy-policy
```

## Files

- `privacy-policy.md` - Markdown version (for GitHub rendering)
- `privacy-policy.html` - Standalone HTML version (prettier, for Chrome Web Store)

## Update Instructions

1. Edit `privacy-policy.md` or `privacy-policy.html`
2. Update the "Last Updated" date
3. Commit and push changes
4. GitHub Pages will auto-update within a few minutes

## For Chrome Web Store Submission

Use this URL in your Chrome Web Store submission:
```
https://[your-username].github.io/github-notification-manager/privacy-policy.html
```
