## üé• Recording Instructions

### Recommended Tools

**macOS:**
- **Kap** (free): https://getkap.co - Best for GIFs
- **QuickTime Player** (built-in): Screen recording, export as MP4
- **KeyCastr** (free): Show keyboard shortcuts on screen

**Windows:**
- **ScreenToGif** (free): https://www.screentogif.com
- **OBS Studio** (free): https://obsproject.com
- **Carnac** (free): Show keyboard shortcuts

**Cross-platform:**
- **Loom** (free): https://loom.com - Easy recording + editing
- **CloudConvert**: Convert video to GIF (https://cloudconvert.com)

### Recording Settings

```
Resolution: 1280x800 (landscape)
Frame Rate: 30 FPS (60 FPS if tool supports)
Export Format: GIF or MP4
File Size: Keep under 5 MB for GIF, 10 MB for video
Quality: High (lossless or minimal compression)
```

### Preparation Checklist

- [ ] Close unnecessary browser tabs
- [ ] Set browser window to 1280x800 (or record full screen and crop)
- [ ] Prepare dummy notifications (use test data or real GitHub)
- [ ] Practice each scene transition (smooth mouse movements)
- [ ] Enable keyboard visualizer (KeyCastr/Carnac)
- [ ] Clean desktop background (minimal distractions)
- [ ] Test audio if adding narration (optional)

### Recording Tips

1. **Slow down mouse movements** - Smooth and deliberate
2. **Pause between scenes** - 1-2 second pause between major transitions
3. **Use zoom/crop** - Extension is 400px wide, zoom in for clarity
4. **Add text overlays** - Use video editor to add scene descriptions
5. **Test export** - Preview GIF/video before finalizing
6. **Optimize file size** - Use tools like **ezgif.com** to compress
